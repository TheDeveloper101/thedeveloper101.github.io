This artifact is a simple lisp compiler that I wrote in Haskell while attending AmeriHac 2026. I had a ton of fun, and despite not knowing any Haskell going into it, I used this project as a way to learn the language. Additionally, I wanted to learn more about WASM and how it works. In order to do that, I built a Wasm backend for the lisp compiler in Haskell. I learned more about Haskell as well as about Wasm and how it works! The code can be found [here](https://github.com/TheDeveloper101/haskell-learning), and I wrote a detailed blogpost about my experience [here](https://thedeveloper101.github.io/posts/2026/02/amerihac-experience/).
